# sistema
Projeto - sistema para uma loja de cosméticos, joias, perfumes e acessórios.

Requisitos funcionais
RF01 - O sistema deve permitir a autenticação de usuários.
RF02 - O sistema deve permitir gerenciamento de produtos.
RF03 - O sistema deve permitir o gerenciamento de dados de clientes.
RF04 - O sistema deve enviar notificações de pagamentos em atraso. 
RF05 - O sistema deve permitir o gerenciamento de vendas.
RF06 - O sistema deve permitir o gerenciamento de promissórias. 
RF07 - O sistema deve permitir o gerenciamento de histórico de compras de cada cliente.

Requisitos não funcionarios 
RNF01 - O sistema deve possuir interface simples e intuitiva.
RNF02 - O sistema deve estar sempre disponível.  
RNF03 - O sistema deve suportar aumento de produtos e usuários sem perda de desempenho.
RNF04 - O sistema deve garantir integridade das informações cadastradas.
